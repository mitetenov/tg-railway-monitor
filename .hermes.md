# tg-railway-monitor

Hermes: when working in this repo, follow these rules.

tg-railway-monitor is a Telegram bot (v0.1.0) that monitors Georgian Railway ticket
availability and notifies users when seats become available.  Built on
python-telegram-bot 20.x with Python 3.11.

## Git
- Push to **origin** (`mitetenov/tg-railway-monitor`)
- Create PRs against origin's main branch only

## Environment
- Copy `.env.example` → `.env` and set `BOT_TOKEN`
- Per-chat config is managed entirely through bot commands; no ROUTE/DATE/CLASS env vars

## Build
- Syntax check: `python -m compileall . -q -x '.venv|__pycache__|tests|.worktrees'`
- Run all tests: `pytest -v`

## Tests
- All tests live under `tests/` (currently 416 tests)
- Async tests use `@pytest.mark.asyncio` (pytest-asyncio 1.4)
- No pytest config file — defaults are fine
- `check_syntax.py` is a non-pytest standalone syntax checker
- Tests that touch config files may fail with `EACCES` when run as root (e.g. `test_save_to_readonly_raises`)

## Docker
- `docker build --build-arg VERSION=$(cat version.txt) -t mitetenov/tg-ticket-monitor:$(cat version.txt) -t mitetenov/tg-ticket-monitor:latest .`
- `docker compose up -d`

## Architecture (key modules)

| Module | Purpose |
|--------|---------|
| `bot.py` | Telegram bot: `/start` wizard (ConversationHandler, 5 states) + `/stop` + `/lang` inline keyboard |
| `poller.py` | Background poller: per-chat state diff, pause/resume, purchase URL generation |
| `ticket_monitor.py` | Legacy standalone monitor (sync, threading-based) |
| `api.py` | Public API: factory (`get_ticket_api`) + backward-compat module-level aliases |
| `_api_base.py` | `TicketApi` ABC with 5 abstract methods + shared constants |
| `api_tre.py` | `TreGeApi(TicketApi)` — tkt.ge scraper, search_trips, build_purchase_url |
| `i18n.py` | Translation class, user language detection/storage, station name translation |
| `stations.py` | Master station data (Tbilisi, Batumi, Kutaisi, etc.) with RU/KA localised names |
| `config_manager.py` | Per-chat JSON config persistence |
| `utils.py` | format_time, fmt_duration helpers |
| `patch_slots.py` | Python 3.13 compatibility patch for PTB |
