# tg-ticket-monitor

Hermes: when working in this repo, follow these rules.

## Git
- Push to **origin** (mitetenov/tg-railway-monitor)
- Create PRs against origin's main branch only

## Build
- `python -m compileall . -q -x '.venv|__pycache__|tests'` to verify syntax
- Run tests: `pytest -v`
- Docker image: local only (`tg-ticket-monitor:VERSION` in docker-compose.yml)
